# 05_運用手順書

## ユーザー追加

本サンプルではseedデータを編集してユーザーを追加します。本番ではID管理基盤と同期します。

## ユーザー無効化

ユーザー管理画面の無効化ボタンでデモ表示を確認します。本番では状態変更APIと監査ログを連動します。

## MFA再登録

ユーザー詳細からMFA再登録を開始する想定です。実環境では本人確認後に再登録します。

## ポリシー変更

ポリシー管理画面で条件とアクションを確認し、変更時は影響範囲を試験します。

## 認証ログ確認

認証ログ画面でユーザー、アプリ、結果、期間を確認します。

## アカウントロック解除

本人確認、原因調査、失敗回数リセット、再発防止を行います。

## 障害時一次対応

影響範囲、発生時刻、対象ユーザー、対象アプリ、ログ有無を切り分けます。
