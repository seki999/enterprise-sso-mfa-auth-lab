# 03_構築手順書

## 前提条件

Docker Desktop、Node.js、Python 3.12を利用します。

## Docker起動

```bash
cp .env.example .env
docker compose up --build
```

## DB初期化

バックエンド起動時にseedが自動投入されます。手動投入は `cd backend && python -m app.seed` です。

## フロントエンド起動

```bash
cd frontend
npm install
npm run dev
```

## バックエンド起動

```bash
cd backend
python -m venv .venv
.venv\Scripts\pip install -r requirements.txt
uvicorn app.main:app --reload --port 8000
```

## 初期ユーザー確認

`employee01 / Password123!` でログインします。

## Mockログイン確認

ログイン後、MFA画面で `123456` を入力するか承認ボタンを押します。

## スクリーンショット取得

```bash
cd frontend
npm run screenshots
```

## Duo OIDC設定方法

`.env` にClient ID、Client Secret、Issuer、Redirect URIを設定し、`AUTH_MODE=duo_oidc` に変更します。実値はコミットしません。
